# The MSF Project #
这是一个基于Spring Boot的微服务框架

## 使用说明 ##

Maven 依赖中添加：

            <dependency>
                <groupId>cn.com.git.msf</groupId>
                <artifactId>msf-dubbo-starter</artifactId>
            </dependency>
            <dependency>
                <groupId>org.springframework.boot</groupId>
                <artifactId>spring-boot-starter</artifactId>
            </dependency>

## 示例 ##

见 `msf-sample-dubbo` 模块
