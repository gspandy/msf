# The redis-cluster starter module of msf project #
这是msf框架快速添加redis-cluster的模块

## 使用说明 ##

Maven 依赖中添加：

            <dependency>
                <groupId>cn.com.git.msf</groupId>
                <artifactId>msf-redis-cluster-starter</artifactId>
            </dependency>

## 示例 ##

见 `msf-samples-redis-cluster` 模块
